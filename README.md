# # 🔒 HAGANAR - Full-Stack AI Chat Assistant & Glassmorphism Portal

A modern, responsive, and visually striking full-stack web application featuring an advanced **Glassmorphism UI**, secure login gate validation, and a live conversation module seamlessly powered by an Express backend server and the **Google GenAI SDK (Gemini)**.

## 🚀 Features
* **Live AI Agent Integration:** Communicate dynamically with a custom HAGANAR AI agent context pipeline via backend server endpoints.
* **Glassmorphism UI:** Translucent containers featuring sophisticated webkit backdrop-filter layers and dynamic blur depth.
* **Responsive Layout Design:** Fully centered interface structures driven by CSS Flexbox properties that scale perfectly across multiple viewport breakpoints.
* **Custom Branding:** Polished visual design complete with customized favicon mapping and consistent typography rules.
* **Animated Context Elements:** Engaging background atmosphere featuring automated particle-generating algorithms.

## 🛠️ Tech Stack
* **Frontend:** HTML5 (Semantic Structures), CSS3 (Glassmorphism, Multi-stop Gradients, Flexbox Layouts), JavaScript (Asynchronous Fetch API, DOM Manipulation)
* **Backend Runtime:** Node.js (v24+)
* **Server Framework:** Express.js (Endpoint routing and static fallback middleware)
* **AI Integration:** Google GenAI SDK (`@google/genai`)
* **Environment Configuration:** Dotenv system for secure API key isolation
* **Version Control:** Git/GitHub

## 📦 Local Installation & Setup

1. **Clone the project repository workspace:**
\`\`\`bash
git clone https://github.com/manoharrajusarikonda331-mani/Html-File.git
cd "Html File"
\`\`\`

2. **Install necessary node package dependencies:**
\`\`\`bash
npm install
\`\`\`

3. **Configure Environment Parameters:**
Create a secure `.env` file in the root project directory level and populate your access token:
\`\`\`env
GEMINI_API_KEY=your_actual_google_gemini_api_key_here
\`\`\`

4. **Boot Up the Application Server:**
Launch the Node runtime listener inside your terminal workspace:
\`\`\`bash
node server.js
\`\`\`

The backend environment will report active status on port 3000. Open http://localhost:3000 inside your browser view to interact with HAGANAR live!
